Featured Projects

  • KukuFiti

    KukuFiti is a broiler management backend designed to help poultry farmers track critical performance metrics such as Feed Conversion Ratio (FCR) and flock mortality. Deployed on Render, this backend is built with FastAPI and PostgreSQL, utilizing raw SQL queries via asyncpg for fast analytics. Database-level CHECK constraints and triggers ensure complete business logic integrity at the storage layer.

    Read KukuFiti Case Study

  • ZealSync — WiFi & Fiber ISP Billing Platform

    A multi-tenant SaaS ISP billing platform integrating M-Pesa Daraja STK Push, MikroTik RouterOS auto-provisioning via WireGuard VPN, and a four-interface React 19 frontend. FastAPI modular monolith backend with raw SQL via asyncpg; idempotency enforced at the database layer via UNIQUE constraints. Eight cross-tenant isolation threats systematically tested and closed.
